Projects

CS4SF: A Scalable Model for Preparing K-12 Teachers to Provide Rigorous, Inclusive Computer Science Instruction

CS4SF aims to create and validate a transferable, scalable model for preparing K-12 CS teachers to create an inclusive learning environment that engages and challenges all students to learn CS and to provide teachers a support system to facilitate that. This is a collaboration with San Francisco Unified School District.


Three-dimensional Non-intrusive Real-time Fall Detection System

This project will investigate the feasibility of symbiotically integrating WiFi-enabled motion detection and floor vibration detection from Civil Engineering to accurately detect human falling and sequential injury severity in real-time without the need of user intervention. This is a collaboration with Dr. Zhaoshuo Jiang from Civil Engineering at SFSU.


Uncover the Ghosts: Deep Learning based Spam Detection on Social Networks

This project aims to develop a new framework that leverages the differences in dissemination between benign and malicious messages to achieve accurate and efficient spam detection in online social networks with text mining, social network analysis, and deep learning.


OPSWAT Training and Internship Program

This program provides training and internship opportunities to CS students through OPSWAT, which enables students to gain valuable practical work experience and mentoring prior to graduation and prepares them for the full-time jobs.
